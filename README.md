# Strona Wizytówkowa Agencji Marketingowej

Zaawansowana, ale minimalistyczna strona internetowa agencji marketingowej B2B. Zbudowana w oparciu o precyzyjne wytyczne projektowe.

## Technologie
- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React

## Do skonfigurowania
Skopiuj plik `.env.local.example` jako `.env.local` i ustaw parametry do Resend lub SMTP:
```bash
cp .env.local.example .env.local
```

### Uruchamianie
```bash
npm run dev
```
Otwórz http://localhost:3000 w przeglądarce by wyświetlić stronę.
